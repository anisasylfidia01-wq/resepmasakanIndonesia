### 2. Deskripsi Aplikasi
Aplikasi ini adalah katalog resep masakan Indonesia yang terintegrasi dengan Firebase Realtime Database. Pengguna dapat melihat berita bumbu dapur, resep populer, dan mencari olahan berdasarkan kategori daerah.

* **Aplikasi Rujukan**: [Cookpad Indonesia](https://play.google.com/store/apps/details?id=com.grandjayastore.resepmasakan)

### 3. Fitur Utama (Ceklist)
- [X] **Firebase Integration**: Mengambil data secara real-time dari Firebase.
- [x] **Horizontal Scrolling**: Implementasi RecyclerView untuk resep populer dan kategori.
- [x] **Detail Resep**: Menampilkan bahan, alat, dan cara memasak dengan Glide untuk loading gambar.
- [x] **SharedPreferences**: Fitur riwayat untuk mendeteksi resep yang sudah pernah dilihat.
- [x] **Admin Dashboard**: Fitur khusus untuk menambah atau mengelola data masakan.
- [x] **Filter Kategori**: Menampilkan olahan spesifik berdasarkan daerah yang dipilih.

### 5. Cara Menjalankan Aplikasi
* ** Login menggunakan email/google, jika belum punya akun klik belum punya akun untuk masuk ke halaman beranda.
* ** Pada tampilan beranda ada icon bumbu masakan, resep populer, dan khas nusantara, pencarian, profil, buat atau tambahkan resep.
* ** Ketika icon bumbu masakan diklik akan menampilkan gambar bumbu dan penjelasannya.
* ** Ketika salah satu gamabr icon resep populer diklik akan menampilkan bahan, alat dan cara membuat masakan tersebut.
* ** Ketika salah satu gamabar icon khas nusantara diklik akan menampilkan beberapa masakan khas nusantara misalnya masakan khas jawa, dan jika di klik salah satunya akan menampilkan alat, bahan dan cara membuatnya.
* ** Ketika icon pencarian di klik kita bisa mencari masakan yang kita mau, tapi jika tidak ada maka tidak akan menampilkan apapun, tapi jika ada maka kita akan di tampilkan resep yang kita cari itu
* ** Ketika icon profil diklik maka akan ada tampilan nama pengguna dan email pengguna serta tombol untuk logout
* ** Icon userdan tambah yang berwarna kuning itu adalah icon untuk menambahkan resep, menyimpan dan menghapus resep.
* ** Jika resep berhasil disimpan maka akan ada notifikasi bahwa resep itu berhasil disimpan.
* ** Icon logout ketika diklik bisa langsung keluar.
